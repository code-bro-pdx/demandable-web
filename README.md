# Demandable Web (DW) Monorepo

Welcome to the Demandable Web (DW) repository. DW is a decentralized ecosystem connecting homeowners with vetted home service professionals through AI-assisted matching, self-custodial identity/wallets, and smart-contract-based escrow settlements.

## Directory Structure

- `apps/`
  - `web/`: Next.js/Vite Web App for homeowners and professionals.
  - `smart-contracts/`: Hardhat/Foundry codebase for Ethereum L2 smart contracts.
- `packages/`
  - `ai-engine/`: Python and Node.js AI matcher and scope validation agents.
